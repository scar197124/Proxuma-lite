# 🛡️ Proxuma Lite

> **A privacy-first link scanner that works without sending your data anywhere.**

---

## 🔍 What is Proxuma Lite?

Proxuma Lite is a lightweight, privacy-focused URL analysis tool designed to help users evaluate links **before opening them**.

Unlike most tools, Proxuma Lite does not rely on cloud processing.

> **All analysis happens locally. Your data stays with you — always.**

---

## ⚙️ Features

- 🔗 Analyze URLs safely before opening  
- 🧠 Detect suspicious patterns and structures  
- ⚠️ Identify potential phishing indicators  
- 📊 Provide clear risk explanations  
- 🔐 Fully offline-first design  

---

## 🔐 Privacy First

Proxuma Lite is built on one principle:

> **Your data should never leave your control.**

- ❌ No tracking  
- ❌ No background requests  
- ❌ No automatic uploads  

---

## ⚠️ Limitations

- Does not guarantee 100% detection  
- Does not replace antivirus software  
- Does not use live threat databases  

---

## 🚀 How to Use

1. Open the scanner  
2. Paste a URL  
3. Review the risk level and explanation
